# About Alphecca Tools

This is the official GitBook documentation for [Alphecca Tools](https://alphecca.io/en).

Alphecca is an all-in-one Web3 token tool platform supporting Solana and EVM chains. Create tokens, manage liquidity pools, run volume bots, multi-send, and more — no coding required.

### Supported Chains

**Solana**

* Token Creation (SPL Token)
* Bundler (Pump.Fun, Raydium Launch Lab, Let's Bonk)
* Liquidity Pool Create / Add / Remove / Burn
* Volume Bot / Market Making
* Multi Sender / Multi Collector
* Token Management (Freeze, Mint, Update Metadata, Revoke Authority)
* Rent Recovery

**EVM (8 Chains)** Ethereum, BNB Chain, Polygon, Avalanche, Arbitrum, Base, Optimism, Monad

* ERC-20(BEP-20) Token Creation
* Liquidity Pool Create / Add / Remove / Burn
* Volume Bot
* Multi Sender
* Token Settings (Tax, Anti-Bot, Anti-Whale, Blacklist, Mint, Pause, Revoke Ownership)
